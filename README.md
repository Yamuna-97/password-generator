# Password Generator (Python)

This is a simple **Password Generator application** created using **Python and Tkinter**.  
It generates a random and secure password based on the length entered by the user.

## Features
- Simple GUI using Tkinter
- User can enter password length
- Generates strong random passwords
- Uses letters, numbers, and special characters

## Technologies Used
- Python
- Tkinter
- random module
- string module

## How to Run
1. Install Python
2. Run the file using:
   ```bash
   python password_generator.py
